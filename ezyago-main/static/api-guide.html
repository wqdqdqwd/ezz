Binance API Anahtarı Nasıl Alınır?

Ezyago botunuzu kullanmak için Binance API anahtarı almanız gerekiyor. Bu rehber size adım adım yol gösterecek.

1 Binance Hesabınıza Giriş Yapın

Binance.com adresine gidin ve hesabınıza giriş yapın. Eğer hesabınız yoksa, önce bir hesap oluşturmanız gerekiyor.

Önemli: Futures işlemleri için hesabınızda KYC (kimlik doğrulama) tamamlanmış olmalıdır.
2 API Management Sayfasına Gidin

Sağ üst köşedeki profil simgenize tıklayın ve "API Management" seçeneğini seçin.

Alternatif olarak doğrudan şu linke tıklayabilirsiniz: API Management

3 Yeni API Anahtarı Oluşturun

"Create API" butonuna tıklayın.

API anahtarınız için bir isim girin (örn: "Ezyago Bot").

4 İzinleri Ayarlayın

API anahtarınız için aşağıdaki izinleri seçin:

  • ✅ Enable Futures - Futures işlemleri için gerekli
  • ❌ Enable Spot & Margin Trading - Gerekli değil
  • ❌ Enable Withdrawals - GÜVENLİK İÇİN ASLA SEÇMEYİN!
Güvenlik Uyarısı: "Enable Withdrawals" seçeneğini asla işaretlemeyin! Bu, hesabınızdan para çekilmesine izin verir.
5 IP Beyaz Listesi Ekleyin (Önerilen)

Ekstra güvenlik için aşağıdaki IP adreslerini beyaz listeye ekleyin:

216.24.57.1
216.24.57.4

Bu IP adreslerini "Restrict access to trusted IPs only" bölümüne ekleyin.

Güvenlik Artışı: IP kısıtlaması ile API anahtarınız sadece Ezyago sunucularından kullanılabilir.
6 API Anahtarlarını Kopyalayın

API anahtarı oluşturulduktan sonra:

  • API Key - Bu anahtarı kopyalayın
  • Secret Key - Bu anahtarı da kopyalayın (sadece bir kez gösterilir!)
Dikkat: Secret Key sadece bir kez gösterilir. Mutlaka güvenli bir yere kaydedin!
7 Ezyago'ya Anahtarları Ekleyin

Ezyago panelindeki "API Anahtarları" bölümüne gidin ve:

  • API Key alanına kopyaladığınız anahtarı yapıştırın
  • API Secret alanına secret anahtarını yapıştırın
  • "Kaydet" butonuna tıklayın
Tebrikler! Artık botunuzu başlatabilirsiniz!
